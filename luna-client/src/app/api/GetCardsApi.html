cardsRandomGet

URL

https://rws-cards-api.herokuapp.com/api/v1/cards/random

HTTP Method

GET

Response Type

CardList

Parameters

  • Query: n Integer

    If specified and between 1 and 78, returns n number of unique random cards. Else, returns all cards in random order.

cardsSearchGet

URL

https://rws-cards-api.herokuapp.com/api/v1/cards/search

HTTP Method

GET

Response Type

CardList

Parameters

  • Query: q String

    Search all for text (case insensitive)

  • Query: suit String

    Returns only cards of specified suit. e.g. search?suit=wands

  • Query: value value

    Returns only cards of specified value, e.g. search?value=4

  • Query: type String

    Returns only cards of specified type.
